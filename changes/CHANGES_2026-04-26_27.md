# Changes 26-27.04

Brief running log for changes made on 26-27 April 2026.

## 2026-04-26 to 2026-04-27

## PR title
Test flow resilience, UX polish, and recording/mic flow refinement

## Summary (2-3 bullets)
- Stabilized loading, question transitions, and audio behavior to reduce stalls, stale states, and unintended autoplay or overlap.
- Iteratively refined test UX (age badge, pause/resume, onboarding navigation, dev controls, and comprehension scoring/tap behavior) for clearer, more predictable behavior.
- Simplified and hardened microphone/recording flow from login through sound check and in-session pause/resume, while preserving resume state and improving error feedback.
